O Oracle MySQL Cloud Service oferece:

  • MySQL Enterprise Edition
    O MySQL Cloud Service oferece todos os benefícios do MySQL Enterprise Edition para fornecer os mais altos níveis de desempenho, segurança e tempo de atividade do MySQL.

  • Fácil de Usar
    O MySQL Cloud Service oferece um console simples e fácil de usar, baseado na web, para gerenciar suas instâncias do MySQL Cloud. Uma API REST e uma ferramenta de linha de comando também estão disponíveis para administradores.

  • Provisionamento de autoatendimento
    Crie bancos de dados MySQL pré-configurados, otimizados para desempenho, e o cloud tooling automatiza o gerenciamento do ciclo de vida da instância de banco de dados.

  • Escalabilidade elástica
    O escalonamento automatizado permite que os usuários escalem recursos de computação e armazenamento elasticamente, bem como réplicas do MySQL.

  • Alta disponibilidade
    A replicação do MySQL e o monitoramento do MySQL Replication melhoram o tempo de atividade do aplicativo e minimizam as interrupções no serviço.

  • Segurança em múltiplas camadas
    O MySQL Cloud Service oferece recursos de segurança de nível empresarial, incluindo controle de acesso à rede, MySQL Enterprise Firewall, MySQL Enterprise Authentication, MySQL Enterprise Encryption e Transparent Data Encryption e MySQL Enterprise Audit.
  • Backup e Recuperação
    • O MySQL Cloud Service suporta backups quentes, online, recuperação pontual e instantâneos sob demanda.

  • Ferramentas MySQL Best in Class
    • Use ferramentas MySQL familiares e melhores, incluindo o MySQL Enterprise Monitor, o Query Analyzer eo MySQL Workbench, para monitorar e gerenciar o desempenho e a disponibilidade de todas as suas instâncias do MySQL.

  • Gerenciamento unificado de nuvem
    • Os usuários do Oracle Enterprise Manager podem gerenciar todo o seu ambiente Oracle a partir de um "único painel".

  • Integração com o Oracle PaaS e IaaS
    • O MySQL Cloud Service é integrado ao Oracle Public Cloud nas camadas de infraestrutura e de plataforma, proporcionando uma experiência perfeita para o desenvolvimento e a implantação de aplicativos.

  • Suporte Técnico Direto da Fonte
    • Com o MySQL Cloud Service você obtém suporte técnico do MySQL diretamente dos especialistas, que são apoiados pelos engenheiros que desenvolvem os produtos MySQL. Além disso, o Oracle Premier Support representa uma solução de suporte completa para a infraestrutura de nuvem e problemas relacionados ao MySQL.
